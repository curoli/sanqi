# 三棋（Sanqi）

三棋（Sanqi，意为“第三种棋”或“三棋”）是一种双人策略游戏，其中棋子之间的配合起着核心作用。

两位玩家分别为白方和黑方，轮流行棋，由白方先行。

三棋在一个国际象棋棋盘上进行。每位玩家有十六枚棋子，分别占据靠近自己的一侧的前两排。

![图 1：初始局面](images/sanqi-start.svg "图 1：初始局面")

一次走子需要三枚棋子：一枚为进攻子（即移动的棋子），另外两枚为支撑子（保持不动）。每一枚棋子都可以作为进攻子或支撑子使用。

两枚支撑子确定一个支点，该点正好位于它们之间的中点位置。支点可以位于一个格子的中心、两个格子之间边的中点，或四个格子交汇的角点。

进攻子围绕该支点移动到另一侧，使得支点正好位于起始格与目标格的正中间（见图 2）。起始格与目标格不能相同，即使进攻子一开始正好位于支点上也不例外。

进攻子的移动距离不受限制，只要最终位置仍在棋盘范围内。移动方向也不局限于沿棋盘的边或对角线，可以有多种不同方向。

目标格必须为空，或被对方棋子占据；若为后者，则该棋子被吃掉并从棋局中移除。

![图 2：可能走法示例](images/sanqi-moves.svg "图 2：可能走法示例")

当一方无法进行任何合法走子时判负。例如，当一方剩余棋子少于三枚时就会出现这种情况。此外，如果一方的棋子相互阻挡，导致无法进行任何走子，也会判负。
